# Hotel Booking Website UI

## 📄 Research Paper
HOTEL BOOKING WEBSITE  
Link: https://www.researchgate.net/publication/381582399_HOTEL_BOOKING_WEBSITE

## 🎯 Objective
The objective of this project is to design and implement web interface screens based on the outputs shown in the selected research paper.

## 💻 Implementation
The project includes:
- Home Page
- Hotel Listing Page
- Hotel Details Page
- Booking Confirmation Page

## 🛠 Technologies Used
- HTML
- CSS
